Питання:
Чим принципово відрізняються режими відкриття файлу 'a' (append) та 'w' (write) в Python, і в якому випадку доцільно використовувати кожен з них?

Студент: Пікульов І.Ю.
=========================
Відповідь:
Режим 'w' (write) створює новий файл або перезаписує існуючий, стираючи всі попередні дані.
Режим 'a' (append) відкриває файл для дописування — новий текст додається в кінець без втрати старих даних.
Режим 'w' доцільно використовувати для створення нового файлу або повного оновлення вмісту,
а 'a' — коли потрібно додати нову інформацію, зберігаючи попередню.

Питання для наступного студента:
Які основні методи читання даних з файлу існують у Python (read, readline, readlines)? Чим вони відрізняються між собою?

Студент: [Студент: Могильний Д.В.]
=========================
Відповідь:
Основні методи читання даних з файлу в Python:
1. read() - читає весь вміст файлу як один рядок
2. readline() - читає один рядок з файлу
3. readlines() - читає всі рядки файлу та повертає список рядків

Відмінності між ними:
- read() повертає весь вміст файлу як один рядок
- readline() читає по одному рядку за раз, корисний для великих файлів
- readlines() повертає список, де кожен елемент - окремий рядок з файлу
read() та readlines() завантажують весь файл у пам'ять, тому для великих файлів
краще використовувати readline() у циклі або ітерацію по файлу безпосередньо.
